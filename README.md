# Covid19-Analysis_Brics

COVID-19 data analysis focused on BRICS countries. This project performs data cleaning, exploratory data analysis (EDA), and visualization to study pandemic trends such as total cases and deaths. It includes interactive visualizations and insights generated by Python data science tools. Tech Stack:Python, Pandas, NumPy, Plotly, Seaborn, Matplotlib.

# Objectives
1. Clean and peprocessed the covid 19 dataset
2. filter and analyze data for BRICS Countries
3. Remove the invalid or irrelevant records
4. Explore trends using data visualization

# Dataset
The dataset contains global covid-19 statistics including:
1.Country
2.Confirmed Cases
3.Deaths

After cleaning the dataset was filtered to include only BRICS nation
countries include 
1. Brazil
2. Russia
3. India
4. China
5. South Africa

# Key Insights
some insights observed during the analysis :
1. Brazil and India recorded some of the highest cases among the BRICS nation
2. China maintained relatively lower active case count after intial outbreak
